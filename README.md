🌿 Plant Disease Detection using CNN

📌 Overview

A deep learning project that uses a Convolutional Neural Network (CNN) to classify plant leaf images as healthy or diseased.

---

🚀 Features

- Image classification using CNN
- Data preprocessing & augmentation
- Model training and evaluation
- Simple web app using Streamlit

---

⚙️ Tech Stack

- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Streamlit

---

▶️ Usage

Install dependencies

pip install -r requirements.txt

Train model

python src/train.py

Run app

streamlit run app.py

---

📊 Dataset

PlantVillage dataset (Kaggle)

---

📈 Result

Achieves good accuracy in detecting plant diseases from leaf images.
